# Technology Resource Planning & Execution Dashboard

[English](README.md) | [Español](README.es.md)

## Description
Interactive dashboard developed to analyze the difference between planned and executed working hours within a company's technology area.

The project integrates data from multiple sources, including SQL Server and Smartsheet, in order to evaluate how resources and project assignments were being managed across different technology teams.

The main objective was to identify planning deviations, measure resource dedication by project and area, and provide analytical insights to support decision-making.

## Data Integration
The dashboard consolidates information from different data sources:

### Smartsheet
- Planned hours
- Project assignments
- Resource allocation by team leaders

### SQL Server
- Executed hours registered daily through an internal company application
- Historical operational records
- Project execution tracking

### SharePoint
- Consultation of shared records and files between departments
- Complementary information used to validate and consolidate operational data

Due to structural differences between the different data sources, data transformation, cleaning, and normalization processes were required to correctly relate the information and build a reliable analytical model.

---

## Dashboard Features
- Dynamic filters by:
  - Year
  - Month
  - Category
  - Project
- Interactive search by employee name or category
- Line charts to visualize deviations and trends
- Comparison between:
  - Planned hours
  - Executed hours
  - Percentage variation
- Detailed table by:
  - Employee
  - Project
  - Area
  - Total hours
- Monthly historical analysis over a one-year period
- Simultaneous monitoring of approximately 200 employees

---

## Business Objective
Provide visibility into how technology resources were being utilized and validate whether planning performed by team leaders aligned with actual project execution.

The dashboard enabled:
- Detection of overallocated or underutilized resources
- Identification of planning inconsistencies
- Analysis of dedication by project and area
- Improvement of future planning strategies based on historical data
